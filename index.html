- Hola Enlace -

Bienvenida a la pagina web de Cabello Para el Alma R4 (NO OFICIAL). Aquí podras encontrar varias herramientas que te seran de utilidad durante tu coordinación municipal en Cabello para el Alma, una de las principales herramientas que tendras a tu disposición es la sección BASES DE DATOS, como podras suponer aquí se encontraran todas las bases de datos disponibles en la asociación para que las llenes y así evitaremos tener muchos documentos de excel en nuestros discos duros, ahorrando espacio y tiempo.
A manera de recomendación y para una mejor experiencia te mencionamos que es mejor utilizar la plataforma en una Computadora, aunque la funcionalidad de la plataforma no se vera afectada de ninguna manera, la sección estetica (imagenes, textos, secciones, etc.) si estara afectada notandose imagenes muy grandes, textos cortados por la mitad entre otros errores visuales.

Solo enlaces municipales, coordinador regional y administrativos de Cabello para el Alma. El uso de la plataforma esta limitado para que solo ustedes puedan entrar, tanto es así que para acceder a las bases de datos y su contenido se necesita anteriormente haber iniciado sesión con un correo electronico y ID de Usuario autorizado por el coordiandor regional, de no contar con estos datos o no haber sido autorizado no podras acceder a las bases de datos lo cual sí es una ventaja para mantener la seguridad de la asociación, por lo que te pedimos que no compartas tu correo y contraseña con nadie.

¡Cambios importantes! Con motivo de seguir mejorando la plataforma se han realizado algunos cambios.

Generales
  • Se removieron la secciones de "Eventos" y "Archivos" ya que se piensa dar una mejora funcional y estetica a las secciones antes mencionadas.
  • Se agregaron etiquetas de distintos colores que ayudaran a identificar errores que encuentres y la versión del apartado en donde esta dicho error para que se pueda corregir, porfavor apoyanos reportando todos los errores que encuentres al siguiente E-mail ó Whatsapp agregando una captura y descripción del error.
  • Se agregaron animaciones en pequeñas partes de la plataforma para hacerla mas agradable a la vista, estas animaciones no afectan a la funcionalidad de ningun apartado.

    • Inicio de sesión
    Anteriormente se contaba con un inicio de sesión muy poco visual y con pocas funcionalidades seguiremos mejorando este apartado por lo que esperen pronto una nueva modificación.
  • Se mejoro, haciendolo mas estetico y agradable a la vista.
  • Las funcionalidades visuales se mejoraron y agregaron algunas extras a esta seccción.

    • Inicio
    Anteriormente se contaba con un inicio con funcionalidades nulas, es decir esta sección donde estas leyendo este texto no existia y nisquiera estaba dentro de las ideas incluirla.
  • Como habras notado se agrego esta sección, es un paso inicial por lo que mas adelante esta sección se mejorara y agregaran mas cosas.

    • Bases de datos
    Los datos introducios por los enlaces en los registros puede que se vean afectados al hacer una edición lo que ocaciona volver a llenar los espacios con NaN.
  • La funcionalidad de esta sección sigue siendo totalmente igual a la versión aterior presentada, pero se planea una mejora visual y funcional, haciendo mas facil comprender las bases de datos que se manejan aquí.
  • La base de datos trenzas, cuenta con una actualización manual en el TOTAL que cualquier usuario puede utilizar, anterior mente solo el desarollador podia realizar actualizaciones del total, en un futuro se planea que esto sea automatico.
  • La base de datos Pulseras pasa a una version 2.0, se agrego la opción de descargar los comprobantes de los municipios que ya realizaron sus depositos.
